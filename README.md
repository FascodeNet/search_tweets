# Search Tweets
## 使い方
1. .envファイルを作成

    .env.sampleをご参照ください。


2. fascode_search_tweets.pyを実行

    15秒間隔でツイートされます。
    ログ出力のためrootで実行してください。

Systemd Unitファイルも添付しております。
